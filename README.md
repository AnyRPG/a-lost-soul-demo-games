# A Lost Soul Demo Games

## Introduction

This pack contains demos of the game, A Lost Soul, created using the anyrpg-cc0-fantasy-content-pack assets.

It adds the following features and functionality:
* A Lost Soul Story Demo Game.  A copy of the game, A Lost Soul.
* A Lost Soul Character Demo Game.  A copy of all content from A Lost Soul, but you can start as any faction and character.

## Installation

This addon can be installed via the Welcome Window in AnyRPG.

https://github.com/AnyRPG/AnyRPGCore
